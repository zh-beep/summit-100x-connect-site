# Summit 100x Connect

A modern LinkedIn links site for the Summit 100x team.

## Features

- Clean, modern design with animated transitions
- Responsive layout for mobile and desktop
- LinkedIn profile links with visual indicators
- Dark theme with gradient effects
- Deployed on Vercel for fast performance

## Getting Started

### Prerequisites

- Node.js 18+ or use `uv` for package management
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
# or
uv pip install -r requirements.txt
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Deployment

### Deploy to Vercel

1. Push your changes to GitHub
2. Connect your repository to Vercel
3. Vercel will automatically detect Next.js and deploy

Or deploy with Vercel CLI:

```bash
npm install -g vercel
vercel
```
